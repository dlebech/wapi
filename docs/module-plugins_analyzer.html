Module: plugins/analyzer

A node that consists of a FFT analyzer node that enables analysis of the playing waveform. Provides convenience methods for creating HTML5 canvas elements that display the output of the analysis.

Parameters:
Name Type Description
context AudioContext

The AudioContext to use for the analyzer.
Implements:
  • WapiPlugin

Example
var analyzer = new Analyzer();
var canvas = analyzer.createFrequencyGraph(300, 300);
document.body.appendChild(canvas);
