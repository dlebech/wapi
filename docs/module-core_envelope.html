Module: core/envelope

Creates an Attack Decay Sustain Release envelope (ADSR) for the given AudioNode parameter.

Notice that larger values of time give shorter intervals for the parameter, e.g. a decay time of 10 is shorter than a decay time of 3.

To use purely as attack-decay node, simply leave out the sustain and release parameters.
