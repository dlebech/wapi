Module: instruments/bass

An instrument that plays a classic electric synth bass tone.

Parameters:
Name Type Attributes Description
context AudioContext
options object <optional>
Properties
Name Type Attributes Default Description
sustain boolean <optional>
 false

Indicates whether or not the bass sustains its tone until a noteOff event is received. The default is false which means that the bass has a short and defined tone.
