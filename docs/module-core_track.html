Module: core/track

An audio track. Implements the same methods as an AudioNode so it can be used directly in an audio node network.

The difference between an AudioNode an a track is that a track contains a chain of audio nodes called "plugins" that are controlled within the context of the track.

In this sense, a track is more like a track in a traditional DAW than an AudioNode.

Parameters:
Name Type Description
context AudioContext

The audio context that this track is part of.
