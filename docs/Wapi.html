ADSREnvelope(param, startTime, minValue, maxValue, attack, decay, sustainValue, sustain, release)

Creates an Attack Decay Sustain Release envelope (ADSR) for the given AudioNode parameter. Notice that larger values of time give shorter intervals for the parameter, e.g. a decay time of 10 is shorter than a decay time of 3. To use purely as attack-decay node, simply leave out the sustain and release parameters.
Parameters:
Name Type Description
param AudioParam The audio parameter to create the envelope for, e.g. frequency for a sine node or gain for a gain node.
startTime int The start time for the envelope.
minValue float The minimum value.
maxValue float The maximum or peak value.
attack int The attack time.
decay int The decay time.
sustainValue float The value for the sustain.
sustain int The sustain time.
release int The release time.

drum(time, destination, startFrequency, endFrequency, frequencyDecay, gainDecay)

Creates a synthesized drum sound. Good for drum kicks with varying intensity.
Parameters:
Name Type Description
time int The time that the instrument will start playing.

startFrequency int The start frequency of sine wave for the drum.
endFrequency int The end frequency of the sine wave for the drum.
frequencyDecay int The rate at which the the frequency decays between start and end frequency. Higher numbers mean faster decay.
gainDecay int The rate at which the gain (volume) decays.
